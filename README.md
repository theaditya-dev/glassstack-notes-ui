# glassstack-notes-ui
A modern glassmorphism-based note card stack UI built using HTML, CSS, and JavaScript. Users can create, store, and manage notes with smooth stacked card interactions and localStorage support.
✨ Features:
- Glassmorphism UI design
- Dynamic card creation
- LocalStorage data saving
- Stack card animation (up/down navigation)
- Category-based note system
- Responsive design
